# robotti-imuri-ohjelma
koululle projekti robotti imurin varauksesta

suunnitelmana on luoda verkkosivu jolla olisi kirjautuminen.
kirjautuminen pitää kirjaa nimistä eli kuinka monesti tietty ihminen on tässä vaiheessa varannut robotti imuria.
sivulla olisi myös mahdollisesti viikon mittainen kalenteri joka näyttäisi milloin robotti olisi jo varattu jolloin sitä ei voisi kukaan muu valita
kalenteri myös voisi näyttää kuka imurin on varannut ja mihin tilaan
itse varaaminen tapahtuu lomakkeen kautta johon lisäät milloin haluat imuria varata ja mihin tilaan
sivu olisi puhelin ystävällinen
sivu pitäisi kirjaa lomakkeista eli kuka sitä on varannut kuinka monesti ja mihin tiloihin sitä on varattu kuinka monesti
